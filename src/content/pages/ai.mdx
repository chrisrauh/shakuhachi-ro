# AI Transparency

I'd had the idea for years, but never the time to build it.

Claude Code helped me turn small pockets of time into something real. The reality is that this site wouldn't exist without AI, specifically LLMs.

I know this technology raises valid concerns and I don't dismiss them.

And I want to be transparent about how the site was made. The [source code is on GitHub](https://github.com/chrisrauh/shakuhachi-ro) if you want to see exactly how it all works.

This page follows the [/ai convention](https://slashpages.net/#ai) — a growing practice for websites to be upfront about their use of AI.
