# AI Transparency

I had the idea for shakuhachi.ro for years, but never the time to build it.

Claude Code helped me turn small pockets of time into something real. The reality is that this site would not exist without AI, specifically LLMs.

I know this technology raises valid concerns, and I take them seriously. So I want to be transparent about how the site was made.

The [source code is on GitHub](https://github.com/chrisrauh/shakuhachi-ro) if you want to see how it was written. Almost all of it was generated by AI. In a few places, I wrote the code myself because it was faster or easier than explaining it to the agent.

Surprisingly to me, prompting my way through the development doesn't feel very different from writing the code myself. It feels like another new, if different, tool that helps me do what I am trying to. My goal is to build shakuhachi.ro for others to use, and here it is. Granted, this is a small personal project, so I expect that challenges from using AI may come later.

This page follows the [/ai convention](https://slashpages.net/#ai), a slash page standard for websites to disclose their use of AI.
