Hi there! I'm Chris.

I’m a shakuhachi student and a web developer, and I’m building **shakuhachi.ro**.

It’s a place to discover, create, transform and share shakuhachi scores. It’s inspired by [The Session](https://thesession.org/), Flat.io and vexflow/vextab, but built for shakuhachi.

## Forking

If you’ve used GitHub (or similar tools), “forking” will feel familiar.

If not, you can fork any score on the site to create your own editable copy. Handy when you want to edit, transpose, or adapt a piece.

Your version stays independent. It links back to the original for reference, but your changes won’t affect it. And yes, you can fork the same score multiple times if you want a few different takes. You can fork your own scores.

## Embedding scores

A big reason I'm building this site is the score renderer: a standalone web component for displaying shakuhachi notation on any webpage.

The live example below shows it in action:

<div class="score-example">
  <shakuhachi-score
    class="score-viz"
    columns="1"
    data-score='{"title":"Akatombo (Opening Phrase)","notes":[{"pitch":{"step":"ro","octave":0},"duration":1},{"pitch":{"step":"tsu","octave":0},"duration":1},{"pitch":{"step":"tsu","octave":0},"duration":2,"dotted":true},{"pitch":{"step":"chi","octave":0},"duration":1},{"pitch":{"step":"ri","octave":0},"duration":1},{"pitch":{"step":"ro","octave":1},"duration":1},{"pitch":{"step":"u","octave":1},"duration":1},{"pitch":{"step":"hi","octave":1},"duration":1},{"pitch":{"step":"ro","octave":1},"duration":2}]}'
  ></shakuhachi-score>
</div>

You can embed scores on your own website, blog post, or use the renderer as a building block for your own tool. Learn more in the [GitHub project](https://github.com/chrisrauh/shakuhachi-ro).
