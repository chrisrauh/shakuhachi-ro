Hi, I'm Christian.

I play shakuhachi, and I built this website because I couldn't find anything quite like it online — a place where players could share pieces the same way musicians share playlists.

The project is inspired by two websites I love: [The Sessions](https://thesession.org/), a community hub for Irish traditional music, and [Flat.io](https://flat.io/), a beautifully designed tool for composing and sharing western sheet music. I wanted something in that same spirit, but for shakuhachi.

## How to Use

### Browse Scores

The library on the home page shows all available scores. You can search by title or composer — pretty straightforward!

### Create a Score

Click "Create score" to start a new score. The score renders live as you type, so you can see what it looks like in real time. Give it a title and save to publish it.

### Fork a Score

You might be wondering: what's forking?

Forking creates your own editable copy of someone else's score. It's useful when you want to make personal modifications — like adding your own fingering notes, transposing, or adapting the piece to your level. Your copy is completely independent; changes you make won't affect the original, and you can share your version freely.

To fork: open any score and click "Fork." That's it!

## Notation Formats

The platform supports three input formats:

- **JSON** — The native format. A structured way to describe each note's pitch, octave, and duration. If you're starting fresh, this is what I'd recommend.
- **MusicXML** — The industry standard used by notation software like Finale, Sibelius, and MuseScore. Handy if you're importing existing scores.
- **ABC notation** — A text-based format that's popular in folk and traditional music communities.

All three formats are interchangeable in the editor — switch between them and the content gets automatically converted.

## Open Source

Shakuhachi.ro is open source, and the code is available on [GitHub](https://github.com/chrisrauh/shakuhachi-ro).

The shakuhachi notation renderer is also available as a standalone library, if you want to embed shakuhachi notation in your own projects.

### Using the Renderer

The easiest way is the `<shakuhachi-score>` web component — just include a script tag and drop the element wherever you need it:

```html
<!-- Include the script once in your HTML -->
<script src="https://shakuhachi.ro/embed/shakuhachi-score.js"></script>

<!-- Default: auto-detect columns based on container height -->
<shakuhachi-score data-score='{...}'></shakuhachi-score>
```

For npm projects, you can also import the `ScoreRenderer` class directly:

```typescript
import { ScoreRenderer } from 'shakuhachi-ro';

const renderer = new ScoreRenderer(container);
await renderer.renderFromScoreData({ /* score data */ });
```

### Live Example

Here's the opening phrase of "Akatombo" (Red Dragonfly), a beloved Japanese folk song. This is exactly what it looks like when embedded:

```html
<!-- Include the script once in your HTML -->
<script src="https://shakuhachi.ro/embed/shakuhachi-score.js"></script>

<!-- Default: auto-detect columns based on container height -->
<shakuhachi-score data-score='{...}'></shakuhachi-score>

<!-- Single column with intrinsic height -->
<shakuhachi-score columns="1" data-score='{
  "title": "Akatombo (Opening Phrase)",
  "style": "kinko",
  "notes": [
    { "pitch": { "step": "ro", "octave": 0 }, "duration": 1 },
    { "pitch": { "step": "tsu", "octave": 0 }, "duration": 1 },
    { "pitch": { "step": "tsu", "octave": 0 }, "duration": 2, "dotted": true },
    { "pitch": { "step": "chi", "octave": 0 }, "duration": 1 },
    { "pitch": { "step": "ri", "octave": 0 }, "duration": 1 },
    { "pitch": { "step": "ro", "octave": 1 }, "duration": 1 },
    { "pitch": { "step": "u", "octave": 1 }, "duration": 1 },
    { "pitch": { "step": "hi", "octave": 1 }, "duration": 1 },
    { "pitch": { "step": "ro", "octave": 1 }, "duration": 2 }
  ]
}'>
</shakuhachi-score>

<!-- Explicit column count (6 columns) -->
<shakuhachi-score columns="6" data-score='{...}'></shakuhachi-score>
```
