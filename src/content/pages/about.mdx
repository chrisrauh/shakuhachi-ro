# Hi there! I'm Chris.

I’m building **shakuhachi.ro**.

I’m a shakuhachi student and web developer, and I wanted a simple way to share scores online, with friends or on r/shakuhachi, and to connect different versions of the same song.

**shakuhachi.ro** is a place to discover, edit and share shakuhachi scores. It’s inspired by [The Session](https://thesession.org/), [Flat.io](https://flat.io) and [vexflow](https://github.com/0xfe/vexflow)/[vextab](https://github.com/0xfe/vextab), but built specifically for shakuhachi.

## Forking

If you’ve used GitHub or similar tools, **forking** will feel familiar.

If not, forking just means creating your own copy of the score.

Forking is useful when you want to edit, transpose, or adapt a piece without losing the original. Your new version stays independent. It links to the original for reference, but your changes won’t affect it.

You can also fork the same score multiple times. It's a great way to try variations.

## Embedding scores

One big reason I'm building this site is the score renderer... I wanted to drop a shakuhachi snippet into a blog post and comment on it directly, but I couldn’t find a good way to do that.

So I built a standalone web component for displaying shakuhachi notation on any webpage.

Here's how you can embed it in your own site:

```html
<script src="https://shakuhachi.ro/embed/shakuhachi-score.js"></script>
<shakuhachi-score
  columns="1"
  data-score='{
    "title": "Akatombo (Opening Phrase)",
    "notes": [
      { "pitch": { "step": "ro", "octave": 0 }, "duration": 1 },
      { "pitch": { "step": "tsu", "octave": 0 }, "duration": 1 },
      { "pitch": { "step": "tsu", "octave": 0 }, "duration": 2, "dotted": true },
      { "pitch": { "step": "chi", "octave": 0 }, "duration": 1 },
      { "pitch": { "step": "ri", "octave": 0 }, "duration": 1 },
      { "pitch": { "step": "ro", "octave": 1 }, "duration": 1 },
      { "pitch": { "step": "u", "octave": 1 }, "duration": 1 },
      { "pitch": { "step": "hi", "octave": 1 }, "duration": 1 },
      { "pitch": { "step": "ro", "octave": 1 }, "duration": 2 }
    ]
  }'
></shakuhachi-score>
```

That’s the idea: scores should be as easy to share as a screenshot... or easier.

If you want to embed scores in your own site or tool, the details are in the [GitHub project](https://github.com/chrisrauh/shakuhachi-ro).
