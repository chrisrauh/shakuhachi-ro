Configuration Options

  • x, y - Starting position for the system
  • columnHeight - Height of each vertical column (default: 500px)
  • columnSpacing - Horizontal distance between columns (default: 80px)
  • notesPerColumn - Maximum notes per column before wrapping (default: 20)
  • formatterOptions - Controls spacing density within columns

API Methods

  • renderColumn(notes, renderer, columnX) - Render single vertical column
  • renderColumns(notes, renderer) - Render multiple columns (right-to-left)
  • renderSingleColumn(notes, renderer, x, y) - Render at specific position
  • getFormatter() - Access the internal formatter for calculations
